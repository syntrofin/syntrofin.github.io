Money Recognizing Methods/RevRec Treatments

Attribute Full Recognition (Point in Time) Daily/Monthly Proration (Over-Time) Milestone-Based (Event-Based) Manual Recognition
How it creates the POB Single milestone; turns "Green" immediately upon triggering event (e.g., Invoice). Set to "Fulfilled" on the Service Start Date. Created in "Pending" (Red Light); waits for external updates like Task completion. Created without automated logic; stays "Not Started."
How it creates the Schedule Single RevenueTransaction for 100% of the value. Uses Start/End Dates to calculate days/months ("Slices"). Remains "Active" but dormant; generates records only on percentage changes. Built, but requires manual user action (button click) to generate transactions.
Accounting Result One SAP Document Number; no monthly "bleeding." Multiple SAP Document Numbers (one per month). SAP entries created only when business milestones are reached. Complete manual control over the SAP BAPI trigger.
